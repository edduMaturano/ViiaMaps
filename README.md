# ViiaMaps

Um aplicativo móvel voltado para a comunidade local, com o objetivo de facilitar o acesso a informações sobre a acessibilidade de estabelecimentos e espaços públicos na região. Inspirado na ideia de um mapa interativo, o app permite que usuários encontrem locais que oferecem recursos como rampas de acesso, banheiros adaptados, vagas reservadas para idosos e cadeirantes, entre outras facilidades.
